# Synthetic Aperture Radar (SAR)

This project is intented to be a source of information and ideas relating to the use of SAR and its applications.

Potential project ideas will also be discussed here in case there is sufficient interest to build online tools allowing users to access detailed SAR information easily.

Radar data can be obtained day or night or under cloud which is often the case during floods.


SAR also has important applications in:
 - Flood monitoring during disasters 
 - Oil spill thickness analysis
 - Crop type analysis
